# Key to Programming

Key to Programming ist ein storybasiertes Konsolen-RPG in C++, das typische Hürden auf dem Weg in die Informatik spielerisch darstellt.

Der Spieler stellt sich den Gegnern Prokrastination, Zweifel, Zeitdruck und
Komplexität und sammelt Items, die Fähigkeiten dauerhaft oder temporär beeinflussen.
Am Ende öffnet sich symbolisch das „Tor zur Informatik“.

---

# Features

- Storybasiertes RPG mit mehreren Gegnern
- Rundenbasierter Kampf mit Angriffen, Buffs & Debuffs
- Item-System (Heilung, Extra-Zug, permanente Buffs)
- ASCII-Grafiken & Animationen in der Konsole
- Textausgabe mit Tipp-Effekt
- Modularer Code mit Klassenstruktur (OOP)

---

# Technischer Überblick

- Sprache: C++
- Paradigma: Objektorientierte Programmierung
- Struktur:  
  - `src/` – Implementierungen  
  - `include/` – Header-Dateien  
- Zentrale Klassen:  
  `Spiel`, `Held`, `Gegner`, `Attacke`, `Item`, `Inventar`, `Display`

---

# Starten

Projekt in Visual Studio öffnen und im Debug-Modus ausführen.

# Ziel des Projekts

Dieses Projekt ist als Lern- und Bewerbungsprojekt entstanden und zeigt:
- saubere Code-Struktur
- logisches Denken
- Kreativität
- Motivation für eine Ausbildung im Bereich Informatik
